# {{ page.title }}

Desenvolvimento do projeto completo de design dos dashboards da Plataforma Nilo Peçanha, contemplando estudos visuais aplicados à visualização de dados no ambiente Microsoft Power BI (PBI), com definição de hierarquia informacional, organização espacial, contraste, legibilidade, padronização cromática e consistência gráfica entre telas, assegurando leitura analítica qualificada, comparabilidade de informações e unidade visual institucional.
